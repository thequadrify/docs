## Começando

Para começar a utilizar a API Quadrify, é essencial entender como realizar a autenticação e fazer chamadas aos endpoints disponíveis.

### Autenticação

Todas as requisições à API devem ser autenticadas utilizando o seguinte cabeçalho HTTP:
```Authorization: Bearer <YOUR_API_TOKEN>```

Substitua `<YOUR_API_TOKEN>` pelo seu token de acesso pessoal.

### Fazendo chamadas à API

Para fazer uma chamada à API, você precisará usar o método HTTP apropriado para o endpoint desejado. A maioria dos endpoints da API Quadrify suporta os métodos `GET`, `POST`, `PUT`, e `DELETE`.
